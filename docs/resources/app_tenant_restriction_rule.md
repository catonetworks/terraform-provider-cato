# cato_app_tenant_restriction_rule (Resource)

Manages a rule in the Cato app tenant restriction policy. Underlying GraphQL is marked @beta; behavior and fields may change.

## Example Usage

```terraform
# App tenant restriction rule (GraphQL @beta).
resource "cato_app_tenant_restriction_rule" "example" {
  at = {
    position = "LAST_IN_POLICY"
  }
  rule = {
    name        = "Example tenant restriction rule"
    description = "Managed by Terraform"
    enabled     = true
    action      = "INJECT_HEADERS"
    severity    = "LOW"
    application = {
      # id   = "office365"
      name = "Office365"
    }
    # headers = [{ name = "X-Example", value = "secret" }] # value is sensitive in schema
    schedule = {}
    source   = {}
  }
}
```

## Import

Import uses the Cato **rule id** as the import `id` (written into `rule.id` in state).

```shell
terraform import cato_app_tenant_restriction_rule.example <rule_id>
```
