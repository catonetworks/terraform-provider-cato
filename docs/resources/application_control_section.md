# cato_application_control_section (Resource)

Manages a custom section in the Cato Application Control (App & Data Inline Protection) policy. Adding a section can reassign existing rules into that section and create a draft revision; see provider documentation. Underlying GraphQL is marked @beta.

## Example Usage

```terraform
# Creating a section can attach existing rules to the new section and open a draft
# policy revision. Review impact in the Cato UI before publish.
resource "cato_application_control_section" "custom" {
  at = {
    position = "LAST_IN_POLICY"
  }
  section = {
    name = "Terraform application control section"
  }
}
```

## Import

Import uses the Cato **section id** as the import `id` (written into `section.id` in state).

```shell
terraform import cato_application_control_section.example <section_id>
```
